WEEK 4: CSS (BASICS) OVERVIEW LEARNING OBJECTIVES I. MeSS with cSS

Maia to do: Lab (in pairs or teams of three)-->create HTML and CSS files that everyone can access (like this: https://benfrain.com/independent-scrolling-panels-body-scroll-using-just-css/). Then have everyone view the HTML in their browser, and one by one comment out a line of the CSS code. Open word and write out what they think each line of CSS code does. Afterwards, study the Guide to Flexbox. Check to see if right/wrong. Come up with questions. Put questions on board. Have people answer each other’s questions. (alternatively: open up a website and mess with cSS and see how things change)
