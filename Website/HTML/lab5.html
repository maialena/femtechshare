WEEK 5: THE GRID
OVERVIEW

We're going to talk about perhaps the most important concept in web design and development: LAYOUT! To start, let's read about 7 Pro Tips when it comes to layout

LEARNING OBJECTIVES
  1. Understand the Grid model and practice implementing different box alignment exercises
  2. Tinker with layouts that you can use for your website
I. Grid By Example

We're going to look at one of my favorite articles of all time: Grid By Example . Seriously. I use this ALL the time -- and so will you! The Grid is magical.

To appreciate the Grid, let's look at what code used to look like when people had to hack contrived ways to get a proper layout going: And now, drumroll please....the code for Grid: Cool, right? Before we dive into the details, there's some things you should know about 'the Grid':
  • Pro's
    1. It's supported by Chrome and a few other browsers (lookup)
    2. It's SUPER easy and convenient to use once you get the hang of it
    3. It makes your CSS Style sheet look beautiful and neat
  • Con's
    1. Many browsers don't use it (lookup)
    2. Because they don't support it, you'll have to provide alternatives (we'll get more into the details of this later)

Yes, there's only one real Con -- but it's kinda a big one. If a Browser doesn't support the Grid, all your pretty formatting won't show up to the user, which is bad news bear. The good news is that a lot of browsers are in the process of developing support for the Grid since it's so popular and 'up and coming.' Thus it is still really valuable for you to learn and master.

First, create a new file in your Share folder (challenge: can you do this from your terminal?). Make a new "grids_practice.html" file. Open it up and start walking through the exercises from Grid By Example above. If you don't understand one, ask for help or skip it for now and come back.

Pro Tip: Try not to copy and paste the provided .html and .css files from the website. You'll learn more if you actually type it out.
