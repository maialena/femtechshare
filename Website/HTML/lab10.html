Overview

Today we're going to have some fun with HTML5 Canvas, which is a tool used to draw 2D graphics on your web page.

Learning Objectives
  1. Draw shapes and text using HTML5 Canvas
  2. Create basic and advanced animations
  3. Transform the canvas grid
  4. Manipulate pixels
Basic Canvas Example
