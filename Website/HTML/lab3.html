WEEK#3: HTML BASICS OVERVIEW LEARNING OBJECTIVES LAB 3 Lecture:

  • Go over Best CSS Practices for Beginners: http://code.tutsplus.com/tutorials/30-css-best-practices-for-beginners--net-6741
  • Start EdX Module 3 (CSS) in class Lab:

    • On terminal, create the web development environment (ie make directories/folders for images, html files, and css files using UNIX commands)

  • As a class review basic HTML doc components and code it for your website !DOCTIME, lang = "en-US", etc., head, body etc. Have people come to the whiteboard and add a line, one at a time
  • Parts of the page: header, tagsetc.
  • Syntax (everyone should open up a text editor at the same time the teacher has text-editor up on his/her window and goes through line-by-line what everything does)
  • Theory of heading tags: readability -- a good web page is modeled after a well-written book
  • Discussion of go-to resources to use when come into a problem; discussion of how important it is for coders to learn how to learn early on (ie looking things up that they don’t know in coding blogs, etc.
  • Individually, open up style.css sheet and start playing around with the fonts/colors/background-color, etc. See what changes on the web page by mainpulating margin, padding, etc. Homework: Continue working on EdX Module 3: “Design Web Page Elements Using Cascading Style Sheets (CSS)”
