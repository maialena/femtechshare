WEEK 7: Intro to JavaScript
OVERVIEW

Today we get to explore the wonderful world of JavaScript. By now your website is looking pretty snazzy. You've got the layout. You've got the style. But perhaps you feel there's something missing....like animation, or a search box, or an interactive form/survey, or give users feedback when they interact with your website in a particular way (like this ). That's where JavaScript comes in.

We'll go into the details together, but here's a brief list of how JavaScript can enhance your website:

  1. Create a sign-up form
  2. Search box with autocompletion features (brings up suggestions based on what you've typed so far -- typing "de" for instance would signal "desert," "delaware," "dentist")
  3. Info that changes when you reload the page, like dates, times, sports match results, weather, stock market prices, etc.
  4. Drop down menu bar
  5. Organization and modification to the DOM (more on this this later)
  6. Animate elements on a page (ex: show/hide info, highlight sections, etc.)

LEARNING OBJECTIVES
  1. Learn what an API is
  2. Understand how JavaScript works with CSS and HTML
  3. Incorporate new script in website
