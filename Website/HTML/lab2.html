OVERVIEW

HTML is the language we use to provide structure to our web pages. If you think of a word document, it's all the different parts of the page that make up its form: header, footer, title, subtitles, main text (or 'body' in HTML), paragraphs, etc.) The font, text color, background color, and text size is all styling which is left to CSS (Cascading Style Sheets). We'll leave CSS to next week, but for now let's look at the basic building blocks of our HTML doc

To do so, we're going to go through Chapters 1-5 pages 12-94 from Jon Ducket's HTML & CSS: Design and Build Websites (don't worry -- they're mostly pictures). Create a lab2.html file in your HTML directory, open up your text editor, and follow along as we play with HTML syntax.
