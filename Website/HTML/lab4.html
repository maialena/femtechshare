III. Find The CSS From Within

Now that it's Week 4 it's time for some reflection. We want you to appreciate HOW MUCH you've learned in such a short amount of time. Seriously -- it might not look like a lot, but HTML & CSS are the backbone of almost every website out there. So if you can get these languages down, you're well on your way to being an amazing front end web developer! To show you the power of CSS, we're going to follow along the CSS Zen Garden exercise. Create a new subdirectory in your Share folder (challenge: can you do this from your terminal?) and download the index.html & style.css files from the website.

Partner up and look through both files. First make sure you have a general sense of what's going on -- if you don't, talk through what you think the elements you're unsure about do.

Then test it out! Play with the CSS files and log your findings by writing comments next to the elements you've learned about. Use '/* */' to write comments like so: 

 /* This is a comment. The processor will not render it as code */

If you really want to be the above-and-beyond-student, open up a TextEditor and log all the new things you've learned so that you can use it for reference later.

We'll mess with the css for 20 minutes or so and then come to the board with our findings and any lingering questions.
